## ¿Qué es esto?

Este es el mapa para **Corona de Hielo RC1 Remasterizado v3.1.83 Prueba Pública 4** que se subiría a Internet Archive. Sin embargo, debido a varios problemas, hemos decidido subirlo aquí, a GitHub.

**AVISO DE SEGURIDAD: NO introduzca la contraseña como principal Corona de Hielo RC1 Remasterizado v3.1.82.**

**NOTA DEL SUBIDOR: ¡NO SUBIR A LA PÁGINA WEB EPICWAR.COM NI A LA WARCRAFT 3 MAP DATABASE! ESTE MAPA SE HA PUBLICADO EXCLUSIVAMENTE EN INTERNET ARCHIVE Y GITHUB.**

## ¿De dónde sacaste la idea?
La idea es sencilla: tras la finalización del evento FWC 2025, Flyff Universe anunció que se pondrá en marcha un servidor de pruebas público (PTS), y el registro se encuentra [aquí](https://pts-universe.flyff.com/register). Se requiere el código PTS.

Si desea jugar a este mapa (Corona de Hielo RC1 Remasterizado v3.1.83 Prueba Pública 4), primero deberá resolver las dos preguntas del acertijo y, a continuación, podrá jugar al mapa.

Una vez que haya resuelto las dos preguntas del acertijo, obtendrá la contraseña.

## ¿Cómo conseguir este mapa y dónde encontrar el acertijo?
Para acceder al archivo de mapa que ha sido cifrado, primero debe resolver la pregunta del acertijo.

La primero pregunta del acertijo es: **¿Puedes describir el episodio de Ave Mujica en el que Mortis habla con Mutsumi? ¿Dónde se emitió el episodio?**

La segundo pregunta del acertijo es: **¿Podrías explicar por qué Mutsumi odia a Sakiko? ¿Fue en el episodio? ¿O en el cómic? ¿O lo dijo Mortis?**

Una vez respondida la pregunta del acertijo, recibirás la contraseña. Es la misma razón por la que *Wadanohara and the Great Blue Sea* contiene un archivo ZIP primo. A menos que hayas resuelto el acertijo después de jugar este mapa.

### Obtener la Contraseña
No podrás resolver el acertijo hasta que respondas la pregunta. Después, recibirás la contraseña.

# Requisitos de Instalación
Ingrese de piénsalo bien antes de jugar este mapa después de haber recibido la contraseña como respuesta al acertijo.

**Versión de Warcraft III**: Parche 1.27b (mínimo) hasta 1.31/1.32 (la versión 2.0 no es compatible).

**¿Es compatible el mapa con los gráficos de Warcraft III Reforged?**: No, solo gráficos clásicos.

## Instalación y Extracción
**Para Extracción**

Hay tres archivos que contienen lo siguiente:
* Ice Crown RC1 Public Test 4 v3.1.83.7z: archivo del mapa principal para Warcraft 3.
* PASSWORD.zip: archivo de texto con la contraseña para **Ice Crown RC1 Public Test 4 v3.1.83.7z**.
* READTHIS.TXT: necesario para leer el archivo sobre cómo obtener la contraseña a partir de la pregunta del acertijo.

Abre el archivo **Ice Crown RC1 Public Test 4 v3.1.83.7z** con el administrador de archivos 7-zip. Una vez hecho esto, resolverás el acertijo y encontrarás los archivos Readme y Password.zip.

El archivo **PASSWORD.ZIP** es el archivo de contraseña incluido. Si no está seguro de cómo utilizarlo, lea el archivo **READ.TXT**, así como el archivo **READTHIS.TXT**. No podrá abrir la contraseña hasta que resuelva la pregunta del acertijo.

**Para Instalación**
1. Coloque el siguiente directorio como se muestra: `E:\games\Warcraft III\Maps\` o en la carpeta Descargas de esta manera: `E:\games\Warcraft III\Maps\Download` Sin embargo, si el directorio está instalado en Archivos de programa, de esta manera: `C:\Archivos de Programa\Warcraft III` o si ejecuta 64 bits: `C:\Archivos de Programa (x86)\Warcraft III`
2. Ejecute War3.exe o utilice W3l.exe (Warcraft 3 Loader PVPGN) para iniciar el juego.
3. Selecciona **Un Jugador** (no recomendado debido a los hackers/tramposos) o **Red de Área Local** (recomendado para el modo multijugador sin conexión).
4. Haga clic en el botón **Crear juego** en la parte inferior.
5. Busca el mapa donde dice **Ice Crown RC1 Remastered v3.1.83-pt4** (si aún no lo has extraído).

Opcionalmente, puede hacer clic en **Opciones Avanzadas** para personalizar la configuración, como Control compartido total, Observadores y Visibilidad del mapa. Nuestra recomendación es:
